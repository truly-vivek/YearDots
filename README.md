# YearDots

Visualize your year, one day at a time.

YearDots is a minimalist live wallpaper that helps you track your yearly progress visually using a clean dot-based design.

## Features
- Minimal live wallpaper
- Highlights current day
- Shows yearly progress
- Lightweight & battery friendly
- No ads (v1.0)
- No data collection
